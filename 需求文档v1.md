# 水果小程序需求文档

## 一、项目概述

**项目名称**：水果小程序
**技术栈**：微信小程序 + 云开发
**目标用户**：商家（发布水果） + 普通用户（浏览购买）

---

## 二、角色与权限

| 角色 | 说明 |
|------|------|
| 店主 | 创建店铺的人，拥有所有权限（邀请/移除 Owner、删除店铺） |
| Owner | 被店主邀请的管理者，拥有商品管理权限 |
| 普通用户 | 浏览水果、搜索水果、查看详情 |

### 权限矩阵

| 操作 | 店主 | Owner | 普通用户 |
|------|------|-------|----------|
| 邀请 Owner | ✅ | ❌ | - |
| 移除 Owner | ✅ | ❌ | - |
| 删除店铺 | ✅ | ❌ | - |
| 管理店铺信息 | ✅ | ✅ | - |
| 发布/编辑/删除商品 | ✅ | ✅ | - |
| 管理分类 | ✅ | ✅ | - |
| 浏览水果 | ✅ | ✅ | ✅ |
| 搜索水果 | ✅ | ✅ | ✅ |

## 三、数据模型

### 3.1 店铺表 (shop)

```javascript
{
  _id: ObjectId,
  name: String,           // 店铺名称（硬编码）
  logo: String,           // 店铺Logo（硬编码）
  ownerIds: [String],     // 店主+Owner的openid列表
  creatorId: String,      // 店主openid
  createTime: Date
}
```

### 3.2 分类表 (category)

```javascript
{
  _id: ObjectId,
  shopId: String,
  name: String,           // 一级分类，如"榴莲"
  subTitle: String,        // 二级标题，如"金枕榴莲"（可选）
  description: String,     // 分类描述（可选）
  sort: Number,           // 排序
  createTime: Date
}
```

### 3.3 商品表 (fruit)

```javascript
{
  _id: ObjectId,
  shopId: String,
  name: String,            // 商品名称
  categoryIds: [String],   // 所属分类（可多个，也可为空）
  tags: [String],          // 标签（可多个）：新上/低价/热卖/预售/售罄
  mainImage: String,       // 主图
  detailImages: [String],  // 详情图（可选）
  description: String,     // 商品描述（可选）
  origin: String,          // 产地（可选）
  specs: [{                // 规格列表
    name: String,          // 规格名，如"大果"
    weight: String,         // 重量，如"2-3斤"
    price: Number,         // 价格
    stock: Number          // 库存
  }],
  createTime: Date,
  updateTime: Date
}
```

## 五、功能详情

### 5.1 用户端功能

#### 首页
- 顶部：店铺名称（硬编码）
- 横向分类滑动条：显示所有分类
- 下方水果卡片列表：图片 + 名称 + 价格（最低价）+ 标签
- 右上角搜索图标
- 点击分类 → 跳转分类页
- 点击水果卡片 → 跳转详情页

#### 分类页
- 显示当前分类名称 + 二级标题（如果有）
- 分类描述（如果有）
- 该分类下的水果列表

#### 详情页
- 商品主图（可放大查看）
- 商品名称
- 标签展示
- 规格选择：用户可查看各规格的价格和库存
- 商品描述
- 产地（如果有）

#### 搜索页
- 顶部搜索框（实时搜索）
- 按水果名称匹配
- 结果展示：图片 + 名称 + 价格 + 所属店铺

### 5.2 商家端功能

#### 商家登录
- 微信授权登录
- 首次登录自动创建店铺（成为店主）
- 通过邀请链接进入自动加入店铺（成为 Owner）

#### 商家首页
- 顶部：店铺名称 + Logo（硬编码）
- 主按钮：上传水果
- 商品列表：图片 + 名称 + 规格数 + 标签 + 操作按钮
- 操作按钮：编辑、删除

#### 发布商品
- 商品名称（必填）
- 选择分类（可多选，可不选）
- 选择标签（可多选）
- 上传主图（必填，最多9张）
- 上传详情图（可选）
- 商品描述（可选）
- 产地（可选）
- 规格列表：
  - 添加规格（至少1个）
  - 每个规格：名称 + 重量 + 价格 + 库存

#### 编辑商品
- 同发布页，可修改所有字段

#### 删除商品
- 确认弹窗后删除

#### 分类管理
- 添加分类：一级名称 + 二级标题（可选）+ 描述（可选）
- 编辑分类
- 删除分类（有商品时提示）

#### Owner管理（仅店主可见）
- 当前 Owner 列表
- 生成邀请链接（复制）
- 移除 Owner（店主不可被移除）

## 六、业务规则

| 规则 | 说明 |
|------|------|
| 商家注册 | 微信授权即创建/加入店铺 |
| 邀请机制 | 店主生成邀请链接，被邀请人点击即成为 Owner |
| 标签设置 | 预设标签手动选择，可多选 |
| 商品状态 | 商家手动设置标签，无自动判断 |
| 分类关联 | 商品可属多个分类，也可无分类 |
| 删除确认 | 删除商品/分类需二次确认 |
| Owner 移除 | 店主可移除 Owner，店主自身不可被移除 |
